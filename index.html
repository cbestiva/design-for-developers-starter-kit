Principles & Basics

Principles of visual design

There are 7 main principles of visual design:

  1. Alignment: Keeps edges aligned. This helps determin relationships within a page.

  2. Emphasis: Is attention grabbing. This helps determin the focal point of a page.

  3. Hierarchy: Should be recognizable. This ties in with emphasis.

  4. Consistency: Keeps similar elements presented in similar ways. This helps prevent confusion.

  5. White Space: Helps rest the eyes. This is the spacing between elements of a page.

  6. Balance: All elements have optical equilibrium.

    There are 2 types:

    • Symetrical

    • Asymetrical

  7. Unity: Keeps all elements in agreement. This presents all elements of a page as a cohesive whole.

Grid Systems

12 unit grid

The 12 unit grid system should be used at all times. This helps with algining elements of a page and keeps the page balanced and consistent.

The grid consists of 3 parts:

  1. Unit: the basic building block.

  2. Gutter: the space between each unit.

  3. Column: the units and gutters grouped to form a grouping of content.

Typography

Conveying a website's message

Typography makes up 95% of web design and helps convey a site's message. It consists of typeface and font.

Typeface: a stylistic choice. The font name (i.e. Helvetica)

Font: a set of the typeface. (i.e. Hevlvetica bold 12pt)

Other Definitions to Note

Median: the line where most lowercase letters terminate.

Baseline: the line where most letters sit.

X-height: the distance between the median and baseline. The greater the distance, the more readable.

Ascender: the portion of a character that extends above the median.

Descender: the portion of a character that extends below the baseline.

Counter: an area that is entirely or partially closed by a letter form.

Line height: the distance between baselines.

Line length: the max length of a single line of text before the next word wraps underneath.

Kerning: the spacing between letters.

Letter-spacing: the space between a group of letters.

Type Basics

Serif

  • Emulates handwritten letters

  • Older, more traditional look

Sans-Serif

  • Looks more machine typed

  • More modern look

How to Choose & Pair Typefaces

When choosing a typeface you should ask yourself:

  • What message do I want to tell?

  • What style would help deliver this message?

  • Is this legible?

When pairing typefaces they must contrast eachother, or just stick to one typeface and vary it's weight.

Color

Evoking a particular feeling

Hue: a color/shade discernable by basic colors (red, green, blue, etc.)

Value, shade: the amount of black in a color.

Tint, tone: the amount of white in a color.

Primary colors: yellow, blue, red.

Secondary colors: green, purple, orange.

Neutral colors: black, white, brown – supporting colors.

Complimentary colors: colors that are across from each other on the wheel.

Analogous colors: colors next to each other on the wheel.

Cool Colors (left side):

  • Green, blue, purple

  • Calm, stable, recede on a page

Warm Colors (right side):

  • Yellow, orange, red

  • Energetic, draws attention, more forward on a page

Rules of Thumb

Only use one (sometimes two) dominant colors. Using two should contrast each other.

Support and enhance dominant colors with supporting colors. Adjust the value, tone, brightness, and saturation.

Keep all links and headers of a site the same same color.
